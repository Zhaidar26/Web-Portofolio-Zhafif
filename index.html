Lettura

Aplikasi yang terintegrasi dengan AI untuk membantu para petani hidroponik dalam mengawasi dan merawat tanaman hidroponik

Tentang Proyek

Aplikasi yang saya dan teman-teman saya kembangkan ini memberikan sebuah solusi yang praktis untuk para petani hidroponik yang ada di Indonesia.

Aplikasi dan produk ini sudah di perlombakan dan mendapat juara 3 pada lomba FITCOM 3.0 yang diadakan oleh Universitas Dinamika

Teknologi yang Digunakan

React Node.js MySQL Arduino IDE Convolusional Neural Network (CNN) Python Jupyter

Fitur Utama

• Menampilkan data pH dan nutrisi secara real time

• Kamera untuk mendeteksi penyakit pada tanaman

• Berita-berita yang dapat memberikan sebuah informasi kepada para petani

• Leaderboard untuk data penjualan sayur pada tahun ini.

• Panduan untuk memberikan sebuah solusi apabila tanaman terkena penyakit

Hasil yang Dicapai

• AI sudah dapat mendeteksi berbagai jenis penyakit

• Sudah dilakukan test menggunakan selada dan hasilnya baik

• Data pH dan nutrisi terkirim dengan benar dan secara real time

Curtain Call

Aplikasi Smart Home yang terintegrasi dengan AI yang dapat mengontrol tirai secara otomatis maupun menggunakan suara.

Tentang Proyek

Curtain Call adalah aplikasi Smart Home berbasis AI yang dapat mengontrol tirai secara manual maupun otomatis melalui handphone, AI digunakan untuk melakukan perintah suara menutup dan membuka tirai.

Tirai bisa dikontrol secara otomatis agar sesuai dengan kondisi rumah, apabila rumah terlalu gelap rumah dapat membuka tirai untuk mendapatkan cahaya matahari, atau bisa masuk ke dalam mode sleep yang dimana tirai akan selalu tertutup dan tidak akan terbuka sebelum sleep mode dimatikan.

Teknologi yang Digunakan

React Expo TypeScript MongoDB Arduino IDE Python

Fitur Utama

• Mode otomatis, mode sleep, dan mode kontrol manual

• Mode omotatis dapat membuka tirai sendiri sesuai dengan kondisi rumah

• ada perintah suara untuk menutup dan membuka tirai

• Tampilan yang ramah pengguna

Hasil yang Dicapai

• Pengguna dapat membuka dan menutup tirai melalui perintah suara

• Mode Sleep yang berfungsi dengan baik

• Dalam mode omotatis tirai dapat terbuka dan tertutup sesuai denga kondisi rumah
